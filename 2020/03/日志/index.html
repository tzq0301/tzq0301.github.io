日志logging：

使用日志代替System.out.println("...")，有以下好处：

  1. 可以设置输出样式
  2. 可以设置输出规则，禁止某些级别输出
  3. 可以被重定向到文件，这样可以在程序运行结束后查看日志
  4. 可以按包名控制日志级别

格式参考以下代码：

import java.util.logging.Logger;

    Logger logger = Logger.getGlobal();
    logger.info("...");
